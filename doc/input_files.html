Input Files

up: Parameters file
Keyword Description
RXN_FILE The reactions file (.dat) see reactions_file_format
CONC_FILE The initial concentrations file (_conc.in) see conc_file_format
IN_DIR Path prefix for the reaction and initial concentrations files. Default is "."
PSEUDOISOMER_FILE Database file with delta G0 information for molecules. Default value is pseudoisomer_dg0f.txt. Most users will not want to change this, but may want to add molecules to this file. Please review Pseuodisomer file format .
COMPARTMENT_FILE Database file with compartment size information. File format is one compartment name and its volume per line. A compartment name followed by 1 or more spaces followed by a volume(in liters). If not specified, compartments will all have the default_volume size, as set by the DEFAULT_VOLUME parameter.
SBML_FILE This is only used by the sbml2bo executable to specify the sbml file to be converted to a boltzmann style reactions file. No default value.
MS2JS_FILE Data base file for translating modelseed molecule id's to json molecule id's. Only used by sbml2bo executable. Default value is modelseed_2_json.srt .
