Algorithm parameters and flags

up: Control Parameters
Keyword Description Default
WARMUP_STEPS Number of stochastic boltzmann steps (reaction firings) to take before starting the record loop. When used from biocellion this may as well be 0 as the only difference between warmup and record steps is the printing. 1000
RECORD_STEPS Number of stochastic boltzmann steps (reaction firings) to take while periodically printing out intermediate information about concentrations and likelihoods. 1000
USE_DEQ
1: Before performing warmup steps use an ODE solver to approach steady state.
0: Do not use an ODE solver before the warmup steps.
0
USE_PSEUDOISOMERS
1: Use the pseudoisomer database to compute reaction energies of formation.
0: Use the user supplied energy of formation in the reactions file.
1
USE_DGZERO
1: Use the pseudoisomer database to compute reaction energies of formation only when a delta g0 was not supplied.
0: Always use the pseudoisomer database to compute energies of formation.
0
USE_BULK_WATER
1: Fix the concentration of the solvent moleucles in all compartments.
0: Allow the solvent concentration to vary in some compartments.
1
USE_METROPOLIS
1: Use the metropolis method to accept or reject a reaction choice based on its likelihood.
0: Don't Use the metropolis method to accept or reject a reaction choice based on its likelihood.
1
USE_ACTIVITIES
1: Allow the user to modify the likelihoods of reactions by ACTIVITY/ENZYME_LEVEL lines in the reactions file.
0: Reaction likelihoods are not allowed to be modified, and activities are all set to 1.0.
0
USE_ENZYME_LEVELS Synonymous with USE_ACTIVITIES, they both have the same effect. 0
USE_REGULATION
1: Modify the likelihoods of reactions by up or down regulation as specified by PREGULATION and NREGULATION lines in the reactions file.
0: Do not allow regulation of reaction likelihoods based on the regulation lines in the reactions file.
1
USE_STEADY_STATE
1: Adust the likelihoods of reactions by the steady state adjustoments in compute_kss.c
0: Do not adjust the reaction likelihoods by the steady state adjustments in compute_kss.c
1
MAX_REGS_PER_RXN Maximum number of regulations (P and/or N) allowed to be given per reaction. 4
RSEED0 Fibonacci seed 0 for the first random number generator. 6765109461
RSEED1 Fibonacci seed 1 for the first random number generator. 1771128657
RSEED2 Linear congruential seed for the first random number generator. 4636875025
RSEED3 Fibonacci seed 0 for the second random number generator. 1649015676
RSEED4 Fibonacci seed 1 for the second random number generator. 7568311771
RSEED5 Linear congruential seed for the second random number generator. 5205784363
