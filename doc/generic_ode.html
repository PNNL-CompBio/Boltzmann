Generic ODE method parameters

up: ODE method Parameters

These are only active when the USE_DEQ parameter is set to 1.

Keyword Description Default
ODE_SOLVER_CHOICE
0: ODE23TB from MATLAB.
1: CVODE from SUNDIALS
0
NO_ROUND_FROM_DEQ
0: When converting concentrations from the ode solve back to counts, round them down to integers.
1: When converting concentrations from the ode solve back to counts, do not round them down to integers.
0
ODE_T_FINAL End point in time for the ODE Integration. Usually you want this to be sufficiently long to achieve steady state. 10.0
ODE_STOP_STYLE
TIME Terminate the ODE integration only when ODE_T_FINAL is reached (or an error occurs)
VEC Terminate the ODE integration when the norm of the derivative vector is less than some threshold. See ODE_STOP_NORM and ODE_STOP_REL. (Currently not implemented)
TIME
ODE_STOP_NORM Not yet implemented.
INF Infinity or max norm.
1 The 1 norm = sum of the absolute values of the vector elements.
2 The 2 norm also known as the Euclidian norm = square root of the sum of the squares of the vector elements.
INF
ODE_STOP_REL Not yet implemented.
ABS When VEC is the ODE_STOP_STYLE, stop when the specified norm of the derivative vector is less than ODE_STOP_THRESH.
REL When VEC is the ODE_STOP_STYLE, stop when the specified norm of the derivative vector divided by the corresponding norm of the concentrations vector is less than ODE_STOP_THRESH.
ABS
ODE_STOP_THRESH The stopping threshold for the derivative norm or relative derivative norm when ODE_STOP_STYLE is VEC. 1.0e-8
DERIV_THRESH When computing concentration derivatives, to avoid noisy behavior as steady state is approached if a derivative's magnitude is less than DERIV_THRESH it is set to 0. 1.0e-8
ODE_JACOBIAN_CHOICE Determines how the ODE solver computes the Jacobian.
0 Use the ode_num_jac routine to numerically approximate the Jacobian.
n, n in [1:14] Use an exact formulation of the derivative corresponding to the derivative computation routine lrn_approximate_delta_concs. Currently only implemented for n = 8. Everything else defaults to the numerical approximation.
0
NJ_THRESH Threshold used by ode_num_jac routine to avoid division by 0. Only relevant when USE_DEQ is 1 and ODE_JACOBIAN_CHOICE is 0. 1.0e-6
DELTA_CONCS_CHOICE Determines how the ODE solver computes the concentration derivatives. Currently a value between 1 and 14 see the ODE Derivative functions table. 8
